# BuyConnex Admin

Interface d'administration pour la plateforme BuyConnex, développée avec Angular.

## Prérequis

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm start
```

Le serveur de développement est accessible sur http://localhost:4200 et se recharge automatiquement à chaque modification.

## Compilation

```bash
npm run build
```

Les fichiers de production sont générés dans le répertoire `dist/`.

## Tests unitaires

```bash
npm test
```

Les tests sont exécutés via Karma et Jasmine.
